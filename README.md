# Yelp-Clone

This is a dynamic website with yelp-like features.

          - HTML/CSS
          - Bootstrap
          - JavaScript
          - Node.js & Express.js
          - MongoDB
          - Heroku

# Techniques
              
          - RESTful routing (Index, Show, New, Edit, Create, Edit, Update, and Destroy)
          - Database associations (user, posts, comments, rating)     
          - User Authentication (Passport.js)
          - Google Map API
          - MongoDB 
          - NPM (Middlewares)
          - Code Refactoring
          - Deploy using Heroku
