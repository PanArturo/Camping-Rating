# Yelp-Clone

This is a dynamic website with yelp-like features.

          - HTML/CSS
          - Bootstrap
          - JavaScript
          - Node.js & Express.js
          - MongoDB
          - Heroku

# Techniques
              
          - RESTful routing 
          - CRUD on database system
          - Database associations (user, post, comment, rating)       
          - User Authentication & Authorization (Passport.js) 
          - Flash messages (connect-flash)
          - NoSQL (MongoDB)
          - NPM 
          - Code Refactoring 
          - Deploy using Heroku
