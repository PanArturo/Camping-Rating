# Yelp-Clone

This is a dynamic website with yelp-like features.

          - HTML/CSS
          - Bootstrap
          - JavaScript
          - Node.js & Express.js
          - MongoDB
          - Heroku

# Techniques
              
          - RESTful routing 
          - Database associations (user, post, comment, rating)     
          - User Authentication & Authorization (Passport.js) 
          - Flash message 
          - Google Map API
          - MongoDB 
          - NPM 
          - Code Refactoring
          - Deploy using Heroku
